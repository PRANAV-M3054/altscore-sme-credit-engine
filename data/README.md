\# Data Directory



\## Raw Data (not committed to Git)

Download from Kaggle and place CSV files in `data/raw/`:



\*\*Dataset:\*\* Home Credit Default Risk  

\*\*URL:\*\* https://www.kaggle.com/datasets/mishra5001/credit-card  

\*\*Files needed:\*\*

\- `application\_data.csv` — main loan application records (307,511 rows)

\- `previous\_application.csv` — previous loan history per applicant

\- `columns\_description.csv` — column definitions



\## Simulated Data (generated by script)

GST filing simulation data is generated by running:

```bash

python src/data/simulate\_gst\_data.py

```

Output goes to `data/simulated/gst\_data.csv`



\## Processed Data (generated by pipeline)

Cleaned and feature-engineered data lands in `data/processed/`

after running the notebooks in order.



\## Key Findings from Excel Profiling

\- 307,511 total applicants, 122 columns

\- 23.5% are SME applicants (Self-employed + Businessman + Commercial associate)

\- Default rate: 8.07%

\- Credit-to-income ratio range: 0.14 to 34.92, average 3.95

\- 67 columns have missing values — 20+ columns dropped (>60% missing)
